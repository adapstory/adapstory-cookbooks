## Description

Исследование ценностных предложений позволяет понять, насколько идеи, заложенные в продукт, соотносятся с реально существующими потребностями потребителей.

## Extracted Text

VALUE PROPOSITION RESEARCH

Какая ключевая ценность должна быть в продукте, чтобы он приносил пользу потребителям и достигал целей компании?

Артефакты и документы

Перечень ценностных предложений

Набор характеристик продукта, которые демонстрируют получаемую потребителем ценность. Важно отобразить трассировку этих характеристик на потребности потребителя (проблемы, цели, задачи и т.д.).

Перечень УТП для ценностных предложений

Перечень формулировок ценностных предложений для использования при коммуникации с потребителями, которые в наибольшей степени демонстрируют получаемый им профит.

Модели и инструменты

Value Proposition Canvas

Шаблон описания ценности продукта или бизнес-модели для конкретного сегмента клиента. Для профиля клиента определяются задачи, которые он выполняет в контексте бизнес-модели, его боли и ценности. Со стороны бизнес-модели определяются продукты и сервисы, вокруг которых она строится, ценности, снимающие боли клиентов, и ценности, создающие выгоды. Трассировка левой части шаблона на правую позволяет понять, насколько предложения продукта / бизнес-модели действительно закрывают проблемы и стремления клиентов.

Alexander Osterwalder

Разработка ценностных предложений. Как создавать товары и услуги, которые захотят купить потребители (Остервальдер, Пинье, Бернарда, Смит)

Value Proposition Hypotheses
