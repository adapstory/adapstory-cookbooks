## Description

Поиск масштабируемого способа привлечения потребителей в продукт связан с большим числом экспериментов, чтобы найти верное сочетание свойств аудитории, каналов и офферов, которое позволит предсказуемо привлекать в продукт все большее число потребителей и достичь окупаемости этого привлечения.

## Extracted Text

ACQUISITION CONFIGURATION RESEARCH

Какая конфигурация целевой аудитории, каналов и офферов обеспечит управляемую дистрибуцию продукта?

Артефакты и документы

Документ, дающий полное представление о всех взаимоотношениях между клиентом и компанией: ценобразование для разных сегментов потребителей, позиционирование на рынке, ценностные предложения и свойства продукта (features), процессы продаж, возвратов, пролонгации сделок и т.д. Основной документа для синхронизации разных отделов компании друг с другом.

Описание любых активностей по привлечению потребителей из разных каналов с учетом затрат на production и само привлечение. Должен иметь единую модель атрибуции, учитывать периоды конвертации в клиента, периоды окупаемости, учитывать фактор сезонности и различие воронок активации для разных каналов привлечения.

Модели и инструменты

Модель распределения клиентов по этапам процесса продаж. Хорошо отображает "бутылочные горлышки" в этом процессе. Основные метрики воронки продаж:

количество клиентов на каждом из этапов;

конверсия клиентов с одного этапа на другой;

длительность пребывания клиентов на каждом из этапов.

Метод анализа поведения клиентов или пользователей, объединенных в когорты по временному признаку: установка приложения, просмотр рекламы, совершение покупки и т.д. в одинаковый период времени. Анализ показывает, каким образом для каждой когорты в последующих периодах времени изменяется значение наблюдаемого показателя (например, ROI, retention rate, customer acquisition cost и т.д.). Наиболее эффективен для продуктов с отложенной или повторяющейся конверсией.

Метод верификации показателей продукта за счет тестирования одинаковых версий его функционала для двух или более сегментов пользователей. Пользователи разделяются на сегменты, имеющие одинаковые характеристики (профиль пользователя). Каждому сегменту предлагается одинаковая версия продукта. Определяются показатели, за которыми нужно наблюдать на время эксперимента. Одинаковые значения показателей (с учетом погрешности) подтверждают корректность выборок пользователей для дальнейшего A/B тестирования.

Метод оптимизации показателей продукта за счет тестирования различных версий его функционала для двух или более сегментов пользователей. Пользователи разделяются на сегменты, имеющие одинаковые характеристики (профиль пользователя). Каждому сегменту предлагается своя версия продукта. Определяются показатели, за которыми нужно наблюдать на время эксперимента. По лучшим значениям показателей принимается решение о выборе версии функционала. Важно, чтобы результаты обладали статистической значимостью.

В разделе представлены модели и артефакты для проектирования стратегии привлечения.
