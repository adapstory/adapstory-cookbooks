Awesome Problem Interview Script

Практическое руководство по составлению скрипта для проведения проблемных интервью с показательными примерами.

Почему с пособием стоит ознакомиться:

Руководство содержит конкретные советы о том, каким образом стоит строить вопросы при проведении проблемных интервью, в какой последовательности и каким образом они должны быть сформулированы.

В руководстве приведены три примера составления гипотез проблем по каждому из которых расписан скрипт, чтобы лучше понять каким образом теория должна превращаться в практику.

Скрипт интервью строится от гипотезы проблемы, поэтому в начале приведены инструкции о том, как правильно сформулировать основную и дополнительные гипотезы, и как их результаты исследования превратить в профиль персоны.
