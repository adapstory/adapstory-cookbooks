## Extracted Text

Перечень требований к продукту. Они могут быть сгруппированы по кластерам (например, эпикам). Беклогов может быть несколько: у разных ролей в команде (product, UX, marketing, technical debt, architecture, QA, etc.), по типам фич (out-of-scope, idea, problem, enhancement, etc.), по типам целей (revenue, engagement, retention, etc.).
