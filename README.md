# adapstory-cookbooks

Cookbooks and archived framework materials for Adapstory.

## Product Architecture Framework archive

This repository stores a reproducible archive of public materials from
`https://productframework.ru`, including:

- sitemap-backed page inventory,
- extracted text descriptions,
- linked Miro and Google Docs artifacts,
- downloaded preview/diagram assets,
- generated markdown pages for browsing inside git.

Regenerate the archive with:

```bash
python3 scripts/fetch_productframework.py
```
